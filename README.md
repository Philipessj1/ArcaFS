# ArcaFS

ArcaFS é um gerenciador de arquivos em nuvem desenvolvido em Python com foco em demonstrar conhecimentos em Cloud Computing, AWS e DevOps.

## Tecnologias

- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Docker
- Docker Compose
- React
- Tailwind CSS
- AWS S3
- AWS RDS
- AWS EC2
- GitHub Actions

## Objetivo

Criar uma aplicação semelhante a um mini Google Drive para fins de estudo e portfólio profissional.
